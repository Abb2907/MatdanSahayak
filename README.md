# Bharat Matdan Sahayak 🇮🇳🗳️

**Bharat Matdan Sahayak (India Voting Assistant)** is a modern, responsive web application designed to help Indian citizens navigate the world's largest democratic exercise with absolute clarity. It follows the guidelines of the Election Commission of India (ECI) to simplify the voting process for first-time voters and citizens needing to update their details.

## 🌟 Features

*   **Status Check:** Easily determine the next steps based on whether you possess an EPIC (Voter ID) number.
*   **Forms Guide:** A detailed breakdown of essential ECI forms (Form 6, 6A, 7, and 8) and when to use them.
*   **Voting Day Walkthrough:** A step-by-step visual timeline of exactly what happens at the polling station (ID Check, Inking, Register, EVM/VVPAT operation).
*   **Alternative IDs Checklist:** A clear checklist of the 12 officially accepted alternative ID proofs if you do not have your physical Voter ID card.
*   **Official Verification Guardrails:** Directs users to the official ECI portal (`voters.eci.gov.in`) across all steps for ultimate truth and confirmation.

## 💻 Tech Stack & Design

*   **Framework:** React initialized via Vite.
*   **Styling:** Pure Vanilla CSS.
*   **Design Aesthetic:** Premium dark mode layout utilizing a modern "glassmorphism" style (frosted glass panels). Beautiful gradients inspired by the Indian Tricolor (Saffron, White, Green) are subtly used for highlighting interactions and borders.
*   **Icons:** [Lucide React](https://lucide.dev/) for crisp, clean UI elements.

## 🚀 Getting Started

Follow these instructions to run the project on your local machine.

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/bharat-matdan-sahayak.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd bharat-matdan-sahayak
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```

### Running the Development Server

1.  Start the Vite dev server:
    ```bash
    npm run dev
    ```
2.  Open your browser and visit the URL provided in the terminal (usually `http://localhost:5173`).
